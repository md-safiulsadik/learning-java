# Learning Java

Welcome to **Learning Java**!

This repository is a comprehensive collection of Java learning materials, covering Object-Oriented Programming (OOP), Data Structures & Algorithms (DSA), and competitive programming problems. Whether you are new to Java or looking to strengthen your programming skills, you'll find helpful resources and practical examples here.

## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/md-safiulsadik/learning-java.git
   ```

2. **Navigate to a specific folder:**
   ```bash
   cd learning-java/DSA-Java
   ```

3. **Open in your favorite Java IDE:**  
   Use IntelliJ IDEA, Eclipse, VS Code, or any Java-compatible IDE.

4. **Explore and run the code examples:**
   Each folder contains well-organized code with examples and explanations.

## Prerequisites

- Java JDK 8 or higher
- Basic understanding of programming concepts
- A Java IDE (IntelliJ IDEA, Eclipse, VS Code, or NetBeans recommended)

## Learning Path Suggestion

1. Start with **Object-Oriented Programming** to master OOP concepts
2. Move to **DSA-Java** to learn essential data structures and algorithms
3. Practice with **CodeForce** problems to strengthen problem-solving skills
4. Build confidence with **Projects** for real-world applications
